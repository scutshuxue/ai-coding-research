原理一: Skill 触发机制 — Claude Search Optimization (CSO)

触发链路

Step 1 用户输入消息

|

Step 2 Claude 扫描所有 skill 的 description 字段

匹配关键词: 错误消息、症状、工具名、文件类型

|

Step 3 匹配度 >= 1% 即加载该 Skill 完整内容

|

Step 4 Skill 内容优先级 > 默认 System Prompt

|

Step 5 User 指令最高优先级（可覆盖 Skill）

Frontmatter 规范

---

name: subagent-driven-development

description: Use when executing

implementation plans with

independent tasks

---

Description 的黄金法则

BAD: "dispatches subagent per task

with code review between tasks"

(工作流摘要 -> Claude 走捷径)

GOOD: "Use when executing plans

with independent tasks"

(纯触发条件 -> 正确加载全文)

Description 只描述 WHEN，不描述 HOW

否则 Claude 会从摘要取捷径
